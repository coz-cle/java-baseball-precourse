# 구현 기능 목록

### 게임 준비
- [X] 난수 생성기를 사용하여 상대방(컴퓨터)의 서로 다른 3자리 수 생성

### 게임 진행
- [X] 입력 값 검증 및 변환
  - [X] [예외] 입력 받은 값이 숫자가 아닐 경우
  - [X] [예외] 입력 받은 값이 3자리보다 작거나 클 경우
  - [X] [예외] 입력 받은 값 중에서 중복이 존재 할 경우
- [X] 입력 값과 상대방(컴퓨터)의 3자리 수 비교
- [X] 비교 결과 출력
- [X] 스트라이크 아웃 메세지 출력

### 게임 완료
- [X] 게임 마무리 입력 값 검증
- [X] 게임 종료 이벤트 구현
